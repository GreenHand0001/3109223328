# 项目简介

## 核心功能
- 基于YOLOv8的水下生物检测系统，支持识别海胆、海参、扇贝、海星、水草5类水下生物
- 提供7600张完整标注的数据集
- 支持多种输入源检测：单张图片、图片文件夹、视频文件、本地摄像头
- 界面功能包括：实时显示目标位置、目标总数、置信度等信息
- 支持自定义置信度和IOU阈值设置
- 支持选择显示标签和原图
- 提供多种检测模型选择
- 支持批量检测并查看所有结果
- 支持检测结果保存

## 适用场景
- 水下生物研究与监测
- 海洋生物多样性调查
- 水产养殖生物计数与分类
- 水下生态保护与研究

## 技术特点
- 采用Pycharm/VSCode+Anaconda开发环境
- 技术栈包括Python3.9、OpenCV-Python、PyQt5、Ultralytics、Torch1.9等
- 提供完整源码、训练好的模型、数据集及运行教程
- 配置成功环境可直接使用
